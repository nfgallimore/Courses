This is the README file for the dirvish-stats distribution.

(C) 2013 Sylvain Le Gall

Gather statistics about dirvish vaults.

This program helps to compute some stats about dirvish vaults: - list vaults.
- list images of a vault. - size of an image. - size of extra data between
two images. - date of last image of a vault.

See the files INSTALL.txt for building and installation instructions. 

Home page: http://forge.ocamlcore.org/projects/dirvish-stats/
